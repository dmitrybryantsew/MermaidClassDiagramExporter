# Changelog

## 0.1.0

Initial standalone GitHub packaging of the plugin.

Included:

- Mermaid class diagram export from selection and project folders
- bundled `MermaidDotNet` and `YamlDotNet` dependencies
- UI Toolkit type graph viewer
- layered layout pipeline and routed edge rendering
- focused subgraph navigation with depth-based exploration
- multi-seed focused graph support
- public packaging docs and third-party notices
